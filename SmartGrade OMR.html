How SmartGrade Works

A 5-step image processing pipeline that turns a raw phone photo into a graded result in seconds.

1

📸 Capture the Answer Sheet

The student or teacher photographs the OMR sheet using any standard smartphone. No special setup or lighting required.

2

🔲 Perspective Correction

OpenCV detects the four corner points of the sheet and applies a homography-based perspective transform to straighten and align it.

3

🔍 Bubble Region Extraction & Detection

Each answer row is extracted from the aligned image. Adaptive thresholding and contour analysis identify which bubble has the highest fill density.

4

✅ Answer Key Comparison

Detected responses are matched against the pre-loaded digital answer key to flag correct and incorrect answers per question.

5

📊 Report Generation

An Excel/CSV file is generated with the student's total score and a question-wise performance breakdown, ready to share.

Key Features

Everything built around making OMR evaluation simple, accurate, and accessible for any institution.

📐

Perspective Transformation

Corrects image tilt and skew using OpenCV homography so the sheet is always properly aligned before processing.

🧠

Intelligent Bubble Detection

Pixel density thresholding and contour analysis reliably identify filled bubbles even under variable lighting conditions.

⚡

Instant Grading

Full sheet evaluated in under 2 seconds. Designed to handle bulk exam batches without manual intervention.

📋

Excel / CSV Report

Auto-generates structured reports with scores and question-wise performance suitable for submission or record-keeping.

📱

Smartphone Compatible

Works with any phone camera image — no flatbed scanner, no OMR hardware, no calibration required.

💸

Zero Hardware Cost

Entirely software-based using Python and OpenCV. Any institution with a laptop can deploy it immediately at no cost.
